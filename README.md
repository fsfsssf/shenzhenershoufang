# 🏠 智能房产分析助手

基于大语言模型的二手房智能分析工具，集成乐有家房源数据和百度百科小区信息，为您提供全面的小区点评、房源分析和优选推荐。

![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)
![License](https://img.shields.io/badge/License-CC%20BY--NC--SA%204.0-lightgrey.svg)

## ✨ 功能特点

- 🔍 **小区搜索** - 支持按名称搜索深圳二手房小区
- 🏘️ **AI 小区点评** - 智能分析小区优劣势，生成专业点评
- 🏠 **房源逐套点评** - 批量分析每套房源，评估性价比
- 📊 **成交数据分析** - 展示近12个月成交价格走势
- ⭐ **TOP5 优选推荐** - 三种选房策略（综合性价比/价格敏感/品质优先）

## 🛠️ 技术栈

| 层级 | 技术 |
|-----|------|
| 部署 | Docker 容器 / Flask + Gunicorn |
| 后端 | Python 3.8+ / Flask Web 框架 |
| 工作流 | _run_node() 分发 + ThreadPoolExecutor 真并行 |
| 大语言模型 | DeepSeek (兼容 OpenAI SDK) |
| 数据源 | 乐有家 / 百度百科 |
| 前端 | HTML5 / JavaScript / Chart.js (静态托管 CDN) |
| 数据存储 | 内存字典（任务状态临时存储）

### 文件职责说明

|------|------|

| `leyoujia_skill.py` | 乐有家开放平台 API 封装 |

### 支持的大语言模型

本项目支持任何 **OpenAI SDK 兼容** 的大语言模型，只需配置对应的 `LLM_MODEL_NAME`：

| 模型 | LLM_MODEL_NAME 示例 |
|-----|---------------------|
| DeepSeek Chat | `deepseek-chat` |
| GPT-4 | `gpt-4o` |
| GPT-3.5 | `gpt-3.5-turbo` |
| Claude 3 | `claude-3-opus-20240229` |

## 📖 使用说明

1. **搜索小区** - 在输入框输入小区名称（如"后海花园"）
2. **选择小区** - 从搜索结果中选择目标小区
3. **等待分析** - 系统自动进行小区点评和房源分析
4. **查看结果** - 小区点评和图表先行展示，房源点评逐步加载
5. **TOP5推荐** - 点击按钮选择不同策略的优选房源
